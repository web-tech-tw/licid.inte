# lucid.inte

基於純網頁運行的 [SliTaz](https://www.slitaz.org)/[xPUD](http://www.xpud.org) Linux x86-compatible 虛擬電腦。[前端]

本專案修改自 [LinuxOnBrowser](http://github.com/supersonictw/lob)，為推廣 Linux 而生。

## 專案設定

```
npm install
```

### 編譯且即時重新載入（開發環境）

```
npm run serve
```

### 編譯且壓縮檔案大小（正式環境）

```
npm run build
```

### 檢測並修正檔案（整理程式碼格式）

```
npm run lint
```
